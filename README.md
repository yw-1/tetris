# Tetris
A simple tetris clone in C and SDL2 renderer

![alt text](sc1.png)

## Controls:
* arrow keys: movement
* shift + left/right/down: hard left/right/drop 
* spacebar: hard drop
* p: pause
* Enter: restart
* Esc: quit

For custom size:
`./tetris 30 30 15` 30x30 board with 15x15 pixel squares
